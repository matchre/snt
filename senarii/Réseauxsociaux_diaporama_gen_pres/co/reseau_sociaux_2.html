Du web aux réseaux sociaux

Zoom

Cette vidéo nous donne une définition d'un réseau social :

  • C'est une application qui va utiliser les sciences et technologies de l'information, donc le Web, Internet, les mobiles pour mettre en lien des personnes.

Comment les différencier entre eux ?

  • Par leur usages : LinkedIn : liens par des CV - FaceBook : Échange de messages.

  • Par le type de liens : Réseau d'amis, réseau professionnel, centres d’intérêts.

  • Par les moyens d'y accéder : le web , les mobiles.

  • Par les types de contenus échangés : messages courts, vidéos, images.

Les médias sociaux comme You Tube permettent la production et l'échange de contenus.
